# Deadlines

To run the bot:
* Install `requirements.txt` file:
```
pip3 install -r requirements.txt       
```
* Create `.env` file in the root folder and add:
```
TOKEN="token_taken_from_telegram_chat"
```
* Run the bot locally:
```
python3 bot.py
```

* Find and open the bot in telegram: `@livequeue87_bot`
